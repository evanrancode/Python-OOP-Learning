

This is a Python-based library management system built using object-oriented programming principles. It simulates basic library operations such as tracking users, managing books, and handling borrowing and reservation records.

 Current Features

### `Users` Class
- Generates a unique user ID
- Stores user name
- Tracks borrowed books with due dates
- Tracks reserved books with reservation dates

###  `Books` Class
- Generates a unique book ID
- Stores book title and author
- Tracks book status: `available`, `borrowed`, or `reserved`
- Allows status updates with validation

##  Project Structure

This commit includes:
- `Users` class in `user.py`
- `Books` class in `book.py`

Future updates will include:
- `Library` class to manage system-wide logic
- Integration between users and books
- Borrowing, returning, and reservation workflows
